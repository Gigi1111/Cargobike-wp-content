Advanced Date And Time Range Picker Add-On For GravityForms

== Description ==

This date range picker add-on (gravityforms) creates a dropdown menu from which a user can select a range of dates. I convert javascript plugin called (daterangepicker) to gravityforms custom field.

Features include limiting the selectable date range, localizable strings and date formats, a single date picker mode, a time picker, and predefined date ranges.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/date-and-time-range-picker` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Edit/Create Forms (gravityforms) screen to configure the plugin

== Frequently Asked Questions ==

= can i use this plugin in other form builders? =

no, this plugin only work on gravityforms

= how can i use it? =

go to forms menu (gravityforms),edit / create a form, in "Standard Fields" section you can select "Range Date" field.
